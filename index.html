    Building a Hand-drawn UI System 构建手绘风格UI系统

    2026-03-20 Tutorial # CSS # Design

    Creating a hand-drawn aesthetic for web interfaces requires a combination of SVG filters, carefully chosen fonts, and subtle animations. In this article, we'll walk through the key techniques used to build Craft UI.

    为网页界面创建手绘风格需要结合 SVG 滤镜、精心选择的字体和细腻的动画。在本文中，我们将介绍构建 Craft UI 所使用的关键技术。

    SVG Filters for Wobble Effect SVG滤镜实现抖动

    The core of the hand-drawn look comes from SVG displacement maps. By applying feTurbulence and feDisplacementMap, we can make straight lines appear slightly wobbly.

    "The imperfection is what makes it feel human."

    "不完美正是让它感觉富有人情味的原因。"

    — Design Philosophy

    Filter Chain Explained 滤镜链详解

    The filter chain starts with feTurbulence to generate noise, then uses feDisplacementMap to distort the element edges based on that noise pattern. Adjusting the scale attribute controls the wobble intensity.

    Performance Considerations 性能注意事项

    SVG filters can be expensive to render. Use them sparingly on elements that benefit most from the effect. Consider reducing filter complexity on mobile devices.

    Typography Choices 字体选择

    Choosing the right fonts is crucial. We use a combination of hand-written Chinese fonts and Patrick Hand for Latin characters. The key is maintaining readability while preserving the hand-drawn feel.

    𝓘
    Tip 小贴士

    Always provide fallback fonts for users who may not have custom fonts loaded. 始终为用户提供备用字体。

    Animations & Motion 动画与动效

    Subtle animations bring the design to life. We use CSS keyframe animations for floating, swinging, and pulsing effects. The squiggly animation cycles through multiple SVG filter variations to create continuous hand-drawn motion.

    Micro-interactions 微交互

    Hover states, button presses, and card lifts all contribute to the tactile feel. Each interaction is designed to feel like paper being moved by hand.

    悬停状态、按钮按下和卡片抬起都有助于营造触感体验。每个交互都被设计成像用手移动纸张一样的感觉。

    Summary 总结

    The hand-drawn UI approach works best for personal blogs, portfolios, and creative projects where personality matters more than corporate precision. It's a fun way to make your web presence feel more approachable and human.

    手绘风格的 UI 方法最适合个人博客、作品集和创意项目，在这些场景中个性比企业精确性更重要。这是一种让你的网络形象显得更加平易近人的有趣方式。
